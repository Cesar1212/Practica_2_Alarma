Practica Alarma

El circuito representa una alarma la cual se activa al detectar un campo magnetico
Consta de un led el cual enciende cuando la alarma empeza a sonar, consta de un REEDSWITCH que 
al momento de pasar el IMAN se activa el led verde, y la bocina empeza a sonar, después de un 
tiempo vuelve el ciclo a la normalidad y así sucesivamente cada que se pase el imán.
